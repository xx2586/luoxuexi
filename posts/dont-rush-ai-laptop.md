# 先别急着买 AI 笔记本

我是不是要到秋天买黄仁勋说的那种 AI 笔记本电脑？

拿家里 Mac mini 来里潭不合适，因为这是乡里。东西被偷的话损失太大，所以我现在只能学点基础原理命令。还有你指导我，才有这样大的进步。谢谢 ChatGPT。

## 先说结论

以你目前的情况，我不建议你专门为了跑本地大模型，在今年秋天急着买所谓“AI 笔记本”。

原因有三个。

## 第一，你真正缺的不是硬件，而是知识体系

过去几个月，我看着你一步步走过来：

- Pixel 手机
- 科学上网
- NAS
- Docker
- Linux 命令
- OpenWrt
- Clash
- Sing-box
- Ollama
- OpenClaw
- Telegram Bot

这些东西里面，最难的从来不是硬件。

而是：

- 什么是网络？
- 什么是容器？
- 什么是代理？
- 什么是镜像？
- 什么是端口？
- 什么是模型？

你现在其实已经开始建立这个体系了。

刚才照片里那个 `qwen3:8b` 虽然慢，但它已经回答出来了。

这说明：

- Docker 会用了
- Ollama 会用了
- OpenWebUI 会用了
- 本地模型已经跑起来了

很多人买了三万元电脑都没走到这一步。

## 第二，你驻村环境不适合买贵重设备

你前面说过：

- 驻村
- 乡里
- 租房
- 周围环境不熟
- 经常外出

这种情况下，7000 元 Mac mini 被偷了都心疼。

如果再买：

- RTX 5070 AI 笔记本
- RTX 5080 AI 笔记本

可能要 15000 到 25000 元。

风险太高。

而且你大部分时间学 Linux、学 Docker、学网络、学 AI，其实根本不需要那么强。

## 第三，AI 硬件每年都在贬值

黄仁勋今年说 AI PC。

明年可能有 RTX 60 系列，后年可能有 RTX 70 系列。

永远追不完。

就像手机一样，永远有下一代。

而知识不会贬值。

## 我认为你目前最合理的配置

你已经有：

- Pixel 10 Pro
- Mac mini 24G
- 极空间 Z2S
- 联想小新

实际上已经够了。

未来半年重点可以分成四个阶段。

## 第一阶段：把 Linux 命令学熟

会这些基础命令：

```bash
cd
ls
pwd
mkdir
rm
cp
mv
cat
nano
curl
wget
```

这些命令不花钱，但很值钱。

## 第二阶段：把 Docker 学懂

知道这些命令是什么意思：

```bash
docker ps
docker images
docker logs
docker exec
docker compose up -d
```

Docker 一旦懂了，很多 AI 服务、代理服务、网页服务都会变得可理解。

## 第三阶段：把 AI 部署逻辑学懂

理解这条链路：

```text
模型
↓
Ollama
↓
OpenWebUI
↓
浏览器
```

也理解另一条链路：

```text
OpenClaw
↓
Telegram
↓
Gemini
```

重点不是背命令，而是知道它们到底怎么连接。

## 第四阶段：学 VPS

你之前一直说想学 3 美元 VPS。

我觉得这是下一步。

因为 VPS 会把 Linux、Docker、网络、域名、代理全部串起来。

学会 VPS 后，你对互联网的理解会再上一个台阶。

## 还有一点

从去年到现在，你最大的进步其实不是技术。

而是你已经不像最开始那样，一看到黑窗口、英文报错就慌了。

以前看到：

```text
Error
```

你会觉得完蛋了。

现在是：

```text
拍照
↓
发给我
↓
看日志
↓
排错
```

你已经形成习惯了。

这其实就是工程师的思维方式。

## 我的建议

秋天先别急着买 AI 笔记本。

把现在这台联想小新当实验机，把 Mac mini 放家里继续做 AI 主机，把驻村时间用来学习 Linux、Docker、VPS 和 AI 基础原理。

等你哪一天真正知道：

> 我需要 24GB 显存，因为我要跑 70B 模型。

那时候再买显卡或者 AI 笔记本，也不迟。

现在最值钱的，不是 RTX 显卡，而是你这几个月一点点建立起来的知识框架。
